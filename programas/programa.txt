# Calculadora de Suma Simple Interactiva
# Pide dos números de un solo dígito (0-9), los suma y muestra el resultado.

# --- Constantes ---
# Guarda el valor ASCII de '0' (48) en la memoria para la conversión.
LOADI 48
STORE 251  # Mem[251] = 48 (ASCII de '0')

# --- Lógica del Programa ---
# 1. Pedir el primer número
LOADI 'N'
OUT
LOADI '1'
OUT
LOADI ':'
OUT
LOADI ' '
OUT

# 2. Leer el primer número y convertirlo
IN         # AC = ASCII del dígito
SUB 251    # AC = valor numérico
STORE 250  # Guarda en Mem[250]

# 3. Pedir el segundo número (con un salto de línea)
LOADI 10   # ASCII para nueva línea
OUT
LOADI 'N'
OUT
LOADI '2'
OUT
LOADI ':'
OUT
LOADI ' '
OUT

# 4. Leer el segundo número y convertirlo
IN         # AC = ASCII del dígito
SUB 251    # AC = valor numérico

# 5. Sumar
ADD 250    # AC = suma de los dos números

# 6. Mostrar mensaje de resultado
LOADI 10   # ASCII para nueva línea
OUT
LOADI 'R'
OUT
LOADI ':'
OUT
LOADI ' '
OUT

# 7. Convertir el resultado a ASCII para imprimir
#    (Funciona para resultados de un solo dígito 0-9)
ADD 251    # AC = suma + 48 (convierte el número a su char ASCII)

# 8. Imprimir el resultado
OUT

# 9. Fin del programa
HALT
